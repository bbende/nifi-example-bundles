Description:

A processor that can rename, or exclude, fields from a JSON document. Renaming fields is achieved by providing a comma-separated list mappings from original field name to new field name. Excluding fields is achieved by providing a comma separated list of field names to exclude.

Properties:

In the list below, the names of required properties appear in bold. Any other properties (not in bold) are considered optional. If a property has a default value, it is indicated. If a property supports the use of the NiFi Expression Language (or simply, "expression language"), that is also indicated.

  • Field Mappings
    • A comma separated list of field mappings, i.e. a=x,b=y,c=z.
    • Default value: no default
    • Supports expression language: false
  • Exclude Fields
    • A comma separated list of field names to exclude from the output.
    • Default value: no default
    • Supports expression language: false

Relationships:

  • success
    • A new JSON document with fields applied and/or excluded.
  • failure
    • The original JSON document that failed processing.
